# AI와 함께하는 개발 연습

이 저장소는 AI 어시스턴트를 활용하여 다양한 작은 프로젝트들을 개발하는 연습을 기록합니다.

## AI를 이용한 개발 연습

이 프로젝트는 AI를 이용한 개발 연습을 위한 것입니다. 기능 구현, 버그 수정, 코드 리팩토링을 포함한 개발 과정이 AI 어시스턴트의 도움을 받아 안내되고 실행됩니다. 이 접근 방식은 소프트웨어 엔지니어링 작업에서 AI의 효율성과 능력을 탐색하고 시연하는 것을 목표로 합니다.

## 프로젝트 목록

### 1. 파이썬 테트리스
- **설명:** Python으로 구현된 간단한 테트리스 게임입니다.
- **실행 방법:**
  ```bash
  python3 main.py
  ```
- **조작키:**
    - `a`: 왼쪽으로 이동
    - `d`: 오른쪽으로 이동
    - `s`: 아래로 이동
    - `w`: 블록 회전
    - `q`: 게임 종료

### 2. 숫자 야구 게임
- **설명:** 간단한 숫자 야구 게임입니다.
- **실행 방법:**
  ```bash
  python3 number_baseball.py
  ```

### 3. Rust 테트리스
- **설명:** Rust 언어로 개발 중인 테트리스 게임입니다. (개발 진행 중)
- **위치:** `rust_tetris/` 디렉토리
