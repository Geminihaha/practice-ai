# 테트리스 게임

이것은 Python으로 구현된 간단한 테트리스 게임입니다.

## AI를 이용한 개발 연습

이 프로젝트는 AI를 이용한 개발 연습을 위한 것입니다. 기능 구현, 버그 수정, 코드 리팩토링을 포함한 개발 과정이 AI 어시스턴트의 도움을 받아 안내되고 실행됩니다. 이 접근 방식은 소프트웨어 엔지니어링 작업에서 AI의 효율성과 능력을 탐색하고 시연하는 것을 목표로 합니다.

## 게임 방법

게임을 플레이하려면 `tetris` 디렉토리로 이동하여 `main.py` 파일을 실행하세요:

```bash
cd tetris
python3 main.py
```

## 조작키

*   `a`: 왼쪽으로 이동
*   `d`: 오른쪽으로 이동
*   `s`: 아래로 이동
*   `w`: 블록 회전
*   `q`: 게임 종료
